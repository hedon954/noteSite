    # 1. 用户管理

    # 1.1 添加用户 —— useradd [选项] 用户名

    • useradd 用户名

      会默认创建一个跟用户名同名的组和家目录，并将该用户归入该组中。

    • useradd -d 指定目录 用户名

      给新创建的用户指定家目录

      • 目录若存在：会报错

      • 需要指定一个新的目录：

    # 1.2 指定密码 —— passwd 用户名

    # 1.3 删除用户 —— userdel 用户名

    • 删除用户，但保留家目录 —— userdel 用户名

    • 删除用户及用户家目录 —— userdel -r 用户名

    # 1.4 查询用户信息 —— id 用户名

    • uid：用户id
    • gid：组id
    • 组：组名称

    # 1.5 切换用户 —— su - 用户名

    • 切换到 root 用户 —— su -

    • 切换到低权限用户 —— su - 用户名

      不需要输入密码

    • 切换到高权限用户 —— su - 用户名

      需要输入密码

    • 回退到之前的用户 —— exit

    # 1.6 查看当前用户 —— whoami

    # 1.7 增加组 —— groupadd 组名

    # 1.8 删除组 —— groupdel 组名

    # 1.9 创建用户时指定组 —— useradd -g 用户组 用户名

    创建一个名为 zwj 的用户，这里会自动创建 zwj 同名家目录，并将其指定到 wudang 这个组当中。

    # 1.10 给用户换组 —— usermod -g 新的用户组 用户名

    # 1.11 /etc/passwd 文件

    用户（user）的配置文件，记录用户的各种信息。

    # 1.12 /etc/shadow 文件

    口令的配置文件。

    # 1.13 /etc/group 文件

    组的配置文件，记录 linux 包含的组的信息。
