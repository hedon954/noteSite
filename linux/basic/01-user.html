# 1. 用户管理

# 1.1 添加用户 —— useradd [选项] 用户名

  • useradd 用户名

    会默认创建一个跟用户名同名的组和家目录，并将该用户归入该组中。

  • useradd -d 指定目录 用户名

    给新创建的用户指定家目录

    • 目录若存在：会报错

    • 需要指定一个新的目录：

# 1.2 指定密码 —— passwd 用户名

# 1.3 删除用户 —— userdel 用户名

  • 删除用户，但保留家目录 —— userdel 用户名

  • 删除用户及用户家目录 —— userdel -r 用户名

# 1.4 查询用户信息 —— id 用户名

  • uid：用户id
  • gid：组id
  • 组：组名称

# 1.5 切换用户 —— su - 用户名

  • 切换到 root 用户 —— su -

  • 切换到低权限用户 —— su - 用户名

    不需要输入密码

  • 切换到高权限用户 —— su - 用户名

    需要输入密码

  • 回退到之前的用户 —— exit
