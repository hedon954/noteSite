# 3. Nginx 命令

# 进入 /usr/local/nginx/sbin 目录中

# 1. 查看 nginx 版本号
./nginx -v

# 2. 启动 nginx
./nginx

# 3. 终止 nginx
./nginx -s stop

# 4. 重新加载 nginx
./nginx -s reload
