# JavaSE —— 枚举类

# 1. 枚举类

  • 类的对象只有有限个，且是确定的。
  • 当需要定义一组常量时，建议使用枚举类。
  • 如果枚举类中只有一个对象，则可以作为单例模式的实现的方式。

# 2. 自定义枚举类

# 方式一：JDK5.0 之前，自定义枚举类
/**
 * 枚举类：季节
 */
class Season{

    //1. 属性
    private final String seasonName;
    private final String seasonDesc;


    //2. 私有化类的构造器
    private Season(String seasonName, String seasonDesc){
        this.seasonName = seasonName;
        this.seasonDesc = seasonDesc;
    }

    //3. 提供当前枚举类的多个对象
    public static final Season SPRING = new Season("春天","春暖花开");
    public static final Season SUMMER = new Season("夏天","夏日炎炎");
    public static final Season AUTUMN = new Season("秋天","秋高气爽");
    public static final Season WINTER = new Season("冬天","冰天雪地");

    //4. 提供获取属性的方法
    public String getSeasonName() {
        return seasonName;
    }

    public String getSeasonDesc() {
        return seasonDesc;
    }

    //5. 提供 toString() 方法

    @Override
    public String toString() {
        return "Season{" +
                "seasonName='" + seasonName + '\'' +
                ", seasonDesc='" + seasonDesc + '\'' +
                '}';
    }
}
# 方式二：JDK5.0 之后，可以使用 enum 关键字

# 3. enum 关键字

  • 实例对象必须放在最前面
  • 对象之间用逗号,隔开，最后一个对象用分号;结束
public enum  Season {

    //3. 实例对象
    SPRING("春天","春暖花开"),
    SUMMER("夏天","夏日炎炎"),
    AUTUMN("秋天","秋高气爽"),
    WINTER("冬天","冰天雪地");

    //1. 属性
    String seasonName;
    String seasonDesc;

    //2. 构造函数
    Season(String seasonName, String seasonDesc){
        this.seasonName = seasonName;
        this.seasonDesc = seasonDesc;
    }

}
  • 默认继承 Enum 类
  • toString() 输出如下
        System.out.println(Season.SPRING);                 //SPRING
        System.out.println(Season.SPRING.seasonName);      //春天
        System.out.println(Season.SPRING.seasonDesc);      //春暖花开
        System.out.println(Season.class.getSuperclass());  //class java.lang.Enum

# 4. Enum 类

values()：返回枚举类型的对象数组；

valueOf(String str)：可以把一个字符串转为对应的枚举类对象。要求字符串必须是枚举类对象的名字，如果不是，会有运行时异常；

toString()：返回当前枚举类对象常量的名称。
